Date: Esta columna indica la fecha en la que se registraron los datos de la acción.  
 
Open: Este es el precio de apertura de la acción el día especificado. Es el precio al que se abre la primera transacción del día.  
 
Close: Este es el precio de cierre de la acción el día especificado. Es el precio al que se cierra la última transacción del día.  
 
High: Este es el precio más alto al que se negoció la acción durante el día especificado.  
 
Low: Este es el precio más bajo al que se negoció la acción durante el día especificado.  
 
Adj Close: Este es el precio de cierre ajustado. A veces, las acciones pueden pagar dividendos o realizar acciones corporativas que afecten su precio. El precio de cierre ajustado tiene en cuenta estos eventos para proporcionar una imagen más precisa del rendimiento de la acción con el tiempo.  
 
Volume: Esta columna indica el volumen de acciones que se negociaron durante el día especificado. Es la cantidad total de acciones que se compraron y vendieron durante el día.
